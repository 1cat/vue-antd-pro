# vue-antd-pro
基于vue-cli3 和 [ant-design-vue](https://vue.ant.design/docs/vue/introduce/)搭建的后台管理系统模板，使用json-server做数据mock。

## Project setup

### 数据mock服务启用
```
cd server
```
```
yarn install
```
```
yarn start
```

### 前端服务启动
```
cd ..
```
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### 登录账号
用户名： 任意输入
密码： 字符数字[4-16]位的任意输入

### Compiles and minifies for production
```
yarn run build
```

### Run your tests
```
yarn run test
```

### Lints and fixes files
```
yarn run lint
```

### Run your end-to-end tests
```
yarn run test:e2e
```

### Run your unit tests
```
yarn run test:unit
```
